
# Logistics Delivery Performance Analysis (SQL Project)

This project analyzes logistics shipment data using SQL to identify delivery delays, route inefficiencies, warehouse performance issues, and delivery agent effectiveness.

## Tools Used
- SQL
- MySQL
- Data Analysis

## Dataset Tables
- Orders
- Shipments
- Routes
- Warehouses
- Delivery Agents

## Project Tasks

### Task 1 – Data Cleaning
Remove duplicates, handle missing delay values and standardize date fields.

### Task 2 – Delivery Delay Analysis
Identify routes and warehouses contributing to delivery delays.

### Task 3 – Route Optimization
Evaluate route efficiency using distance and transit time metrics.

### Task 4 – Warehouse Performance
Analyze shipment delays and ranking of warehouses based on on-time deliveries.

### Task 5 – Delivery Agent Performance
Measure on-time delivery rates and identify low performing agents.

### Task 6 – Shipment Flow Analysis
Track shipment status and analyze customer feedback related to delays.

### Task 7 – KPI Reporting
Calculate key performance metrics including on-time delivery percentage and warehouse utilization.

## Key Insights
- Certain routes contribute disproportionately to delays.
- Warehouse congestion increases delivery delays.
- Experienced delivery agents achieve better on-time performance.
- Delivery delays directly impact customer satisfaction.
